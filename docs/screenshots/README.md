# Screenshots

Place application screenshots in this folder so they can be referenced from the main [README](../../README.md) and other docs.

## Naming

Use `<area>-<detail>.png` so captures group naturally:

```text
dashboard-scan.png
rename-preview.png
mux-remux-mp4-conversion.png
track-properties.png
library-overview.png
settings-library-media-servers.png
web-dashboard.png
```

Prefix web UI captures with `web-` to distinguish them from desktop captures.

## Tips

- PNG is preferred; keep individual files under ~1 MB where possible (GitHub renders large images slowly).
- Capture at a normal window size (the desktop app defaults to 1280x800) rather than maximized on a high-DPI display, so text stays readable in the README.
- Avoid captures that show real library paths, server URLs, or API keys.
